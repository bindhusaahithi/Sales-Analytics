Project Overview

This project analyzes retail sales data to identify key business insights related to total sales, profit, regional performance, product categories, and top-performing products.

The project demonstrates a complete analytics workflow using Python for data cleaning and exploratory analysis, SQL for business queries, and Tableau for dashboard-style visualizations.

Tools Used

  • Python
  • Pandas
  • Matplotlib
  • Jupyter Notebook
  • SQL (MySQL)
  • Tableau
  • Git & GitHub

Business Questions

  • What is the total sales revenue and total profit?
  • Which region generates the highest sales?
  • Which product categories perform best?
  • What are the top-selling products?
  • How do sales change over time?
  • How does discount impact profitability?

Key Insights

The West region generated the highest overall sales.
Technology was one of the strongest-performing categories.
A small group of products contributed a large share of total revenue.
Sales trends showed steady activity over time.
Discounting had a noticeable impact on profitability.

Conclusion

This project demonstrates how retail sales data can be transformed into actionable business insights through data cleaning, SQL analysis, and dashboard-style visualization.
