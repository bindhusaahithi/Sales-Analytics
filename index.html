Project overview

A full analytics workflow, packaged as a portfolio case study.

This project analyzes retail transaction data to surface key business insights around sales, profit, regional performance, product categories, and discount strategy.

The work demonstrates practical analyst skills across data cleaning, exploratory analysis, SQL-based business questioning, and dashboard storytelling in Tableau.

Toolkit

Built with analyst-ready tools
