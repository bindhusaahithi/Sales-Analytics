Project Overview

This project analyzes retail sales data to evaluate overall sales performance, category contribution, top products, and regional trends. The objective is to demonstrate a practical analytics workflow used in real business intelligence projects.

Tools Used

  • Python
  • Pandas
  • Matplotlib
  • Jupyter Notebook
  • SQL (MySQL)
  • Tableau
  • Git & GitHub

Dataset

Name: Superstore Sales Dataset

Type: Retail / Business Data

Includes: Orders, products, customers, profit, shipping, and region

Prepared using: Python cleaning workflow

Business Questions

  • What is the total sales revenue and total profit?
  • Which region generates the highest sales?
  • Which categories perform best?
  • What are the top 10 products by sales?
  • How do sales change over time?

Key Insights

West region generated the highest overall sales.
Technology was one of the strongest-performing categories.
A small number of products drove a large share of revenue.
Sales trends showed active month-to-month changes.

Conclusion

This project demonstrates how Python, SQL, and Tableau can be combined to perform real-world sales analysis and communicate insights in a business-friendly way.
