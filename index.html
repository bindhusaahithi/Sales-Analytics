Sales Analytics Project

An end-to-end sales analytics project using Python, SQL, and Tableau to uncover revenue trends, regional performance, category insights, and top-selling products.

View GitHub Repository View Dataset

Project Overview

This project analyzes retail sales data to identify important business insights related to overall sales, profit, regional performance, product categories, and sales trends over time.

Using Python, SQL, and Tableau, the project demonstrates a complete analytics workflow including data cleaning, feature engineering, business querying, and visual storytelling.

This project is designed to reflect a real-world business intelligence workflow and showcase skills relevant to Data Analyst roles.

Tools & Technologies

  • Python
  • Pandas
  • Matplotlib
  • Jupyter Notebook
  • SQL (MySQL)
  • Tableau
  • Git & GitHub

Dataset

Dataset: Superstore Sales Dataset

Type: Retail Sales / Business Data

Includes: Orders, customers, products, sales, profit, shipping, and regions

Prepared Using: Python data cleaning workflow

Project Objectives

  • Analyze overall sales and profit performance
  • Compare sales across regions
  • Identify the best-performing product categories
  • Find the top 10 products by revenue
  • Understand monthly sales trends
  • Demonstrate end-to-end analytics workflow using Python, SQL, and Tableau

Skills Demonstrated

  • Data Cleaning and Preprocessing
  • Exploratory Data Analysis (EDA)
  • Business Querying with SQL
  • Data Visualization
  • Trend Analysis
  • Insight Generation
  • Project Documentation with GitHub

Project Workflow

  1. Data Collection: Obtained the Superstore sales dataset.
  2. Data Cleaning: Cleaned and standardized the dataset using Python.
  3. Feature Engineering: Created year, month, and profit margin columns.
  4. Exploratory Analysis: Analyzed sales, profit, region, category, and product trends.
  5. SQL Analysis: Wrote business queries for revenue, profit, regions, categories, and trends.
  6. Visualization: Built charts and Tableau visualizations for storytelling.

Key Questions Answered

  • What is the total sales revenue and total profit?
  • Which region generates the highest sales?
  • Which category contributes the most to revenue?
  • What are the top 10 products by sales?
  • How do sales change over time?

Key Insights

  • The West region generated the highest total sales.
  • Technology emerged as one of the strongest-performing categories.
  • A small number of products contributed a large portion of revenue.
  • Sales trends showed active month-to-month variation with overall growth.

Why This Project Matters

This project reflects the type of work commonly performed in business intelligence and data analyst roles. It demonstrates the ability to clean real data, explore performance metrics, write SQL queries, create business-focused visualizations, and communicate actionable insights clearly.

Conclusion

This Sales Analytics project demonstrates an end-to-end analytics workflow from raw data to business insights. It combines Python, SQL, and Tableau to provide a structured and practical example of sales data analysis for portfolio purposes.
